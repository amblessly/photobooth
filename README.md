# Snapcrate Photobooth

A browser-based photobooth app with filters, stickers, and photo strip export. No installation required.

---

## Requirements

- A modern browser (Chrome, Edge, Safari, Firefox)
- A working webcam
- Internet connection (for Firebase feedback and Google Fonts)

---

## Getting Started

1. Open `index.html` in your browser
2. Allow camera access when prompted
3. Follow the 4-step flow: **Layout → Capture → Decorate → Save**

---

## How It Works

### 1. Layout
Pick a strip layout and choose how many shots you want (2–6).

### 2. Capture

- Select a countdown timer (3s, 5s, or 10s)
- Optionally pick a filter before shooting
- Click the shutter button to start the countdown
- Use **Retake last** to redo your most recent shot
- Once all shots are taken, proceed to decoration

### 3. Decorate

- Add stickers by tapping any emoji in the sticker panel
- Add custom text via the Text tab
  - Double-tap any text on the strip to edit it
- Set a frame color and optional event banner under the Frame tab

### 4. Save

- Download your strip as **PNG** or **JPG**
- Generate a share code (QR) for easy sharing
- Strips are saved to the in-app gallery for the current session

| Gallery | Photo icon in the header |
